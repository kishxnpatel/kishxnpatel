I focus on reliability, clear fixes, and tidy documentation. Day-to-day that's Windows 10/11, Microsoft 365, AD user/group tasks, imaging (Intune/USB), and common desk-side issues (AV, printers, onboarding, asset updates). I keep ticket hygiene tight (ServiceNow/Jira), write simple KBs, and use light scripting (PowerShell/Python) for inventory/log parsing.
